# RX6800-notifier
.NET Core application that scrapes RX6800 and RX6900 inventory numbers.

This project is based on this Notifier from Niek Nijland > 
https://github.com/NiekNijland/RTX3000-notifier


## Currently scrapes from:
- megekko.nl
- informatique.nl
- cdromland.nl
- coolblue.nl
- cyberport.de
- centralpoint.nl

disabled for now:
- maxict.nl
